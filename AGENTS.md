## Quick Start

1. **Run the installer:**
   - **Linux / macOS / WSL:** `./install.sh` (or `make install`)
   - **Windows (native):** `.\install.ps1`

2. **Select your stack** when prompted:
   - **Python** — agents: Oscar, Scout, Ivan, Jester
   - **TypeScript** — agents: Ostype, Tylead, Tyson, Nova, Marco

3. **The installer will:**
   - Copy agent configurations to `~/.config/opencode/`
   - Install skills
   - Generate `opencode.json`
   - **Copy the correct `AGENTS.md` for your stack** (overwriting this file)
